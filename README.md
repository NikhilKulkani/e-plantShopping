# e-plantShopping

## 🌿 Paradise Nursery Shopping App

This project is a React-based shopping application for a virtual plant store called **Paradise Nursery**.

The application allows users to:

* Browse a variety of houseplants categorized into different groups
* Add plants to a shopping cart
* View and manage items in the cart
* Adjust quantities and view total cost before checkout

## 🛠️ Technologies Used

* React.js
* Redux Toolkit
* Vite
* CSS

## 📄 Features

* Landing page with company details
* Product listing page with categorized plants
* Shopping cart with dynamic updates
* Responsive navigation with cart count

## 🚀 Deployment

The project is deployed using GitHub Pages.
